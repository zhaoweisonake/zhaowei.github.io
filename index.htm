Software Development

  • Wind Turbine Load Extrapolation

    • This program is used to extrapolate 50-year return period extreme loads of wind turbine as IEC 61400-1. Wind turbines - Part1: Design requirements, 3rd edition. 2005. Three weibull parameters distribution for short-term extreme load distribution is used,the scale parameter,the location parameter and the shape parameter are estimated using the L-moment method.

  • DECWT(Dongfang Electric Corporation Wind Turbine)

    • Calculate power curve, thrust coefficient, Cp and AEP of wind turbine at different air density; compare power production for all kinds of schemes for a wind farm, internal use only.

  • WTFA(Wind Turbine Feasibility Analysis)

    • After micro-siting, carry out feasibility analysis for each wind turbine, considering turbulence intensity, wind distribution, air density, etc. factors, currently is under developed.
